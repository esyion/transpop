# 一、PRD（产品需求文档）

## 1. 产品名称

**TransPop（暂定）**

> 一款常驻后台的 AI 翻译工具，通过快捷键快速唤起，提供接近 Raycast 的翻译体验。

---

# 2. 产品背景

目前翻译流程普遍较长：

```
浏览器
↓

打开 DeepL / Google Translate

↓

粘贴文本

↓

等待翻译
```

整个流程需要数十秒。

希望提供一种更轻量的交互：

```
Alt + Space

↓

输入

↓

Enter

↓

得到翻译

↓

Esc
```

整个过程无需切换应用。

---

# 3. 产品目标

### MVP

实现一个桌面端翻译工具：

* 全局快捷键
* 悬浮输入框
* AI 翻译
* 自动识别语言
* 支持目标语言配置

后续再增加：

* OCR
* 划词翻译
* 剪贴板翻译
* AI 写作
